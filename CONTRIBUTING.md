# Contributing

Contributions should improve transparency, safety, docs, site quality, or community usefulness.

Good contributions:

- Better source notes or public metrics refreshes.
- Accessibility and mobile fixes for the site.
- Original meme assets with disclosure.
- Community FAQ improvements.
- Compliance linter improvements.

Not accepted:

- Trading bots or wallet-signing flows.
- Paid listing/boost workflows.
- Fake engagement tools.
- Price predictions, profit claims, or pressure language.
- Anything implying CHATTY is affiliated with OpenAI, ChatGPT, or any company.

Run before proposing changes:

```bash
python3 tools/content_linter.py
python3 dashboard/render_research.py
```
