# CHATTY Do-Not-Post Rules

- Never reply to price bait.
- Never argue with trolls.
- Never quote-post scammers unless warning neutrally.
- Never coordinate raids.
- Never ask for buys.
- Never tag large accounts for attention.
- Never tag OpenAI, ChatGPT, Sam Altman, or unrelated public figures.
- Never post more than one project update per day.
- Never post without disclosure or a link to the disclosure page.
- Never use price targets, return claims, urgency, or certainty.
- Never post screenshots as proof without a source link.
- Prefer zero posts to risky posts.
