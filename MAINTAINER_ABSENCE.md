# Maintainer Absence

CHATTY is designed to remain understandable if the human creator ignores it.

- The public site and repo are canonical.
- Telegram may be unmonitored.
- X may not receive updates.
- Read-only GitHub workflows may refresh static files when safe.
- No wallet, payment, trading, or social posting is automated.
- Report scams or broken links through GitHub Issues if available.

No response from the creator should be interpreted as a promise, endorsement, or financial signal.
