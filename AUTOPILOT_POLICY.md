# CHATTY Autopilot Policy

Status: Dormant Autopilot Mode.

CHATTY is unofficial, not affiliated with OpenAI/ChatGPT, not financial advice, and can go to zero.

Autopilot may:

- Refresh read-only public data snapshots.
- Update static JSON and Markdown transparency files.
- Run content, link, risk, and site checks.
- Generate weekly transparency reports.
- Commit safe repo/site updates through GitHub Actions.

Autopilot must never:

- Post to X or Telegram.
- DM anyone.
- Trade, transfer, burn, lock, bridge, or move crypto.
- Connect wallets.
- Spend money or use paid services.
- Request funds or post donation addresses.
- Create fake engagement.
- Make investment claims or price promises.

If a safe automation cannot complete, it should log the blocker and stop.
